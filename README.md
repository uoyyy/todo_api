# Многопользовательский менеджер задач (Task Manager API)

RESTful API сервис для управления задачами и проектами, разработанный на **FastAPI**. Приложение реализует многопользовательскую архитектуру с изоляцией данных и JWT-авторизацией.

## 🚀 Технологический стек
* **Язык:** Python 3.11
* **Фреймворк:** FastAPI
* **База данных:** PostgreSQL
* **ORM:** SQLAlchemy + Pydantic (валидация данных)
* **Аутентификация:** JWT (JSON Web Tokens), хэширование паролей bcrypt
* **Контейнеризация:** Docker, Docker Compose

## ⚙️ Реализованный функционал
* **Пользователи:** Регистрация и безопасная авторизация (пароли хранятся в виде хэшей).
* **Изоляция данных:** Каждый пользователь имеет доступ только к своим проектам и задачам.
* **Проекты:** Полный CRUD проектов (создание, чтение, обновление, удаление).
* **Задачи:** * Создание задач внутри конкретных проектов.
  * Установка приоритета, сроков (`start_time`, `end_time`) и текстовых заметок.
  * Отметка о выполнении и фильтрация списка задач по статусу (`is_completed`).
